# Leena Saini Portfolio

Static portfolio site built from the resume content in `Resume1.pdf`.

## Quick edits

- Replace the photo by adding `assets/profile-photo.jpg`.
- Update copy, links, and projects in `site-data.js`.
- The resume download button points to `assets/leena-saini-resume.pdf`.

## Run locally

- Open `index.html` directly in a browser, or
- run `python3 -m http.server` inside this folder and visit `http://localhost:8000`.
